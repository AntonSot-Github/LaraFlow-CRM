# LaraFlow CRM

LaraFlow CRM is a simple internal CRM system built with Laravel 12 and PHP 8.4.

This project was created as part of practical Laravel learning and demonstrates backend development skills, including authentication, authorization, and CRUD operations.

## Features

- Authentication system
- User roles (admin / user)
- Clients management (CRUD)
- Deals management (CRUD)
- Tasks management with completion toggle
- Polymorphic comments system
- Authorization using Laravel Policies

## Tech Stack

- Laravel 12
- PHP 8.4
- MySQL
- Blade
- Bootstrap 5
